NancyHighcharts-sample
======================

Using Nancy(OWIN, SelfHost, SuperSimpleViewEngine), Highcharts on Heroku

## Tested environment

 * Windows7 x64
 * .NET Framework 4.5
 * Highcharts 4.0.4
 * jQuery 2.1.3
 * NuGet package
    * Microsoft.Owin.Hosting 3.0.0
    * Microsoft.Owin.Host.HttpListener 3.0.0
    * Nancy.Owin 0.23.2
